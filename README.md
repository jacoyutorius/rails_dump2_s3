# RailsDump2S3
====


## Description

Upload db:dump file to AmazonS3.


## Usage

```bash
export AWS_ACCESS_KEY="Your aws acccess key"
export AWS_SECRET_ACCESS_KEY="Your aws secret acccess key"

rake rails_dump2_s3:dump
```


## Install

```bash
gem install "rails_dump2_s3"
```
